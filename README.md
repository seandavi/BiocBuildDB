# BiocBuildDB

BiocBuildDB is an R package that provides tools for managing and querying
a database of Bioconductor build reports. These reports contain information
about the build status of Bioconductor packages across different platforms and
R versions.

## Installation

You can install the development version of BiocBuildDB from [GitHub](https://github.com/) with:

``` r
# install.packages("BiocManager")
BiocManager("seandavi/BiocBuildDB")
```

## Basic Usage

See the vignettes for detailed examples on how to use BiocBuildDB to query
and analyze Bioconductor build reports.
